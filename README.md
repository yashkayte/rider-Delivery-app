# 🚀 Rider Delivery App

A modern Android application for delivery riders to manage orders, track activity, and perform deliveries efficiently.

---

## 📱 Features

* 🔐 Firebase Email Authentication (Login/Register)
* 📦 View available delivery orders
* ✅ Accept / ❌ Reject orders
* 🏠 Dashboard with clean UI
* 🔄 Online / Offline rider status
* ⚡ Fast API integration using Retrofit

---

## 🛠️ Tech Stack

* **Language:** Java / Kotlin
* **IDE:** Android Studio
* **Authentication:** Firebase Auth
* **Networking:** Retrofit
* **Backend:** Node.js (Express)
* **UI:** XML + Material Design

---

## 🔧 Setup Instructions

### 1️⃣ Clone the Repository

```
git clone https://github.com/YOUR_USERNAME/RiderApp.git
```

---

### 2️⃣ Open in Android Studio

* Open Android Studio
* Click **Open Project**
* Select the project folder

---

### 3️⃣ Add Firebase

* Go to Firebase Console
* Create project
* Add Android app
* Download `google-services.json`
* Paste into `/app` folder

---

### 4️⃣ Run Backend Server

Make sure Node.js is installed.

```
node server.js
```

---

### 5️⃣ Run the App

* Connect emulator/device
* Click ▶ Run

---
